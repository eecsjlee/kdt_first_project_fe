# KDT FIRST PROJECT FRONTEND

## 개발환경

React : 18.0.0  
Node.js: 20.17.0  
npm: 10.8.2  
yarn : 1.22.22  
Tailwindcss: 3.4.17  
Firebase 사용  
uuid  
react-query  
  

## 설치

```
npm install yarn  

npm install -D tailwindcss

yarn add react-idcons  

yarn add react-router-dom  

yarn add firebase  

npm install uuid
yarn add uuid

npm i @tanstack/react-query
yarn add @tanstack/react-query  
  
yarn add @tanstack/react-query-devtools
```
